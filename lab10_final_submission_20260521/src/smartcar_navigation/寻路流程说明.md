# smartcar_navigation

LIMO 的寻路流程包，负责在已有地图上完成定位、路径规划和自动导航。

## 功能说明

这个包实现的是“寻路流程”，核心能力包括：

- 通过 `map_server` 加载静态地图
- 通过 `amcl` 在地图中定位
- 通过 `move_base` 做全局规划和局部避障
- 通过 RViz 发布目标点并观察导航结果

当前保留三种导航入口：

- `smartcar_navigation_teb.launch`
- `smartcar_navigation_dwa.launch`
- `smartcar_navigation_teb_astar.launch`

另外补齐了真实 LIMO 小车的导航入口：

- `limo_real_navigation_dwa.launch`
- `limo_real_navigation_teb.launch`
- `limo_real_navigation_teb_astar.launch`
- `limo_real_amcl.launch`

## 运行流程

### 1. 安装依赖

```bash
sudo apt-get install ros-noetic-navigation
sudo apt-get install ros-noetic-teb-local-planner
```

### 2. 编译工作区

```bash
cd ~/myfile/robote_project/limo_ws
catkin_make
source devel/setup.bash
```

### 3. 启动仿真

```bash
roslaunch limo_gazebo_sim limo_four_diff.launch
```

### 4. 启动一种导航方式

#### TEB 局部规划

```bash
roslaunch smartcar_navigation smartcar_navigation_teb.launch
```

#### DWA 局部规划

```bash
roslaunch smartcar_navigation smartcar_navigation_dwa.launch
```

#### A* 全局规划 + TEB 局部规划

```bash
roslaunch smartcar_navigation smartcar_navigation_teb_astar.launch
```

### 4. 真实小车导航

如果真实车底层已经提供 `/limo/scan`、`odom`、`base_footprint`、`/cmd_vel` 等接口，则可以直接运行：

```bash
roslaunch smartcar_navigation limo_real_navigation_dwa.launch
roslaunch smartcar_navigation limo_real_navigation_teb.launch
roslaunch smartcar_navigation limo_real_navigation_teb_astar.launch
```

如果实车实际使用 `/scan`，则在启动时覆盖：

```bash
roslaunch smartcar_navigation limo_real_navigation_teb.launch scan_topic:=/scan
```

### 5. 在 RViz 中下发目标点

启动后按下面的顺序操作：

1. 用 `2D Pose Estimate` 设置初始位姿
2. 用 `2D Nav Goal` 发布目标点
3. 观察全局路径、局部路径和代价地图

## 默认地图

默认地图文件为：

```text
~/myfile/robote_project/limo_ws/map/map.yaml
```

如果要换地图：

```bash
roslaunch smartcar_navigation smartcar_navigation_teb.launch map_file:=/绝对路径/你的地图.yaml
```

## 文件作用

`launch/amcl.launch`

- 启动 AMCL 定位
- 使用 `/limo/scan`、`odom`、`base_footprint` 和 `map`

`launch/move_base_dwa.launch`

- 使用 DWA 作为局部规划器

`launch/move_base_teb.launch`

- 使用 TEB 作为局部规划器

`launch/move_base_teb_astar.launch`

- 使用 `GlobalPlanner` 作为全局规划器
- 使用 TEB 作为局部规划器

`launch/smartcar_amcl.launch`

- 只启动地图和定位

`launch/smartcar_navigation_teb.launch`

- TEB 导航总入口

`launch/smartcar_navigation_dwa.launch`

- DWA 导航总入口

`launch/smartcar_navigation_teb_astar.launch`

- A* + TEB 导航总入口

`launch/limo_real_amcl.launch`

- 真实车地图定位入口

`launch/limo_real_navigation_dwa.launch`

- 真实车默认全局规划器 + DWA

`launch/limo_real_navigation_teb.launch`

- 真实车默认全局规划器 + TEB

`launch/limo_real_navigation_teb_astar.launch`

- 真实车 A* + TEB

`param/costmap_common_params.yaml`

- 公共代价地图参数
- 定义机器人外形、激光来源和障碍膨胀参数

`param/local_costmap_params.yaml`

- 局部代价地图参数

`param/global_costmap_params.yaml`

- 全局代价地图参数

`param/move_base_params.yaml`

- `move_base` 控制频率、恢复行为和超时参数

`param/dwa_local_planner_params.yaml`

- DWA 专用参数

`param/teb_local_planner_params.yaml`

- TEB 专用参数

`param/global_planner_params.yaml`

- 全局路径规划参数

`rviz/nav_teb.rviz`

- RViz 预设显示配置

`实车导航运行说明.md`

- 真实车运行前提和参数覆盖方式

`package.xml`

- 声明导航流程依赖

`CMakeLists.txt`

- 安装 launch、param、rviz 和 README 文件
